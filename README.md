# eStore (using React)

eStore created with React

**Installion** 

1. npm install
2. npm start

**Features:**

1. Product display (Using fake bike data which is comming from a Rest api.)
2. Quick view  
3. Sorting (Low to high, high to low)
4. Add to cart
    * Delete product from the cart
    * Update quantity (increase/Decrease)
    * Order total (+tax)
5. Product search
