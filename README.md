# eStore (using React)

eStore created with React

**Installion** 

1. npm install
2. npm start

**Features:**

1. Product display (Using fake bike data which is comming from a Rest api.)
2. Quick view  
3. Sorting (Low to high, high to low)
4. Add to cart
    a. Delete product from the cart
    b. Update quantity (increase/Decrease)
    c. Order total (+tax)
5. Product search
